INSTALLATION
    From SimpleSurvival.zip, extract the GameData\SimpleSurvival folder, and place it in your "Kerbal Space Program\GameData" folder.  If the mod has been installed correctly, then you will find the DLL at:

    "Kerbal Space Program\GameData\SimpleSurvival\SimpleSurvival.dll"
